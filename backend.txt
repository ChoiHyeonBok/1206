erp란? 전사적자원관리
ERP(Enterprise Resource Planning, 전사적자원관리)란 기업 내 생산, 물류, 재무, 회계, 영업과 구매, 재고 등 경영 활동 프로세스들을 통합적으로 연계해 관리해 주며, 
기업에서 발생하는 정보들을 서로 공유하고 새로운 정보의 생성과 빠른 의사결정을 도와주는 전사적자원관리시스템 또는 전사적통합시스템

crm란? 고객관계관리
기업이 고객과 관련된 내외부 자료를 분석·통합해 고객 중심 자원을 극대화하고 이를 토대로 고객특성에 맞게 마케팅 활동을 계획·지원·평가하는 과정이다. 

D-ERP : WEB application server에서 erp 프로세스 통합 처리
프론트엔드 백엔드 개발영역 분리
DERP 서버가 DB와 사용자 중간 역할

DEWS Front Designer : 보여지는 화면을 구성하기 위한 디자인 프로그램

메인화면
1) 워크스페이스 탐색기
2) 디자이너 영역 - 기본 9개의 패널 등록 가능 패널에 따라 등록할 수 있는 패널의 개수 상이하다
3) 프로퍼티 영역 - 디자이너 영역에서 특정 컨트롤을 클릭했을 시 우측 프로퍼티 창에 다음과 같이 속성들에 대한 옵션을 설정할 수 있음.
	- 컨버스를 제외한 모든 컨트롤들은 ID(아이디)속성을 가지고 있으며, 자바스크립트에서 컨트롤들을 식별할 때 사용함.
	- change,open,close 이 세 가지 속성들은 JQuery 이벤트 또는 Dews API를 스크립트 상에서 코딩 가능
	- </> 버튼은 스크립트를 Edit할 수 있는 버튼 앞에 파랗게 표시되면 스크립트 이미 작성된 상태
	- 스크립트의 기본 형태
	function() {} 안에 작성

Datasource 영역
Data Provider를 사용 시 @DzApi 정보 파싱 - 찾고자 하는 서비스의 메소드 이름을 검색한 후 선택하면 자동으로 바인딩

상단 버튼을 클릭하고 프로퍼티 창을 보면 다음과 같이 click 이벤트를 스크립트로 작성과 보이기 여부 설정 가능

컨테이너 패널(재무표)
그리드와 폼패널이 합쳐진 패널, 그리드 형태의 패널에서 행을 선택하여 폼패널 형태의 패널에서 값을 조회하거나 변경가능 (우측하단 버튼 클릭 후 split)
분할도 가능하며 분할하면 각 다른 컨트롤들을 배치 할 수 있다.
부모 컨테이너 패널에 속성 설정 시 Select ParentContainerPanel 활성화 후 프로퍼티 값 변경

폼 패널(신상명세서)
다양한 종류의 폼 컨트롤 사용가능, 폼 패널 컨트롤은 라인마다 2개의 컨트롤이 들어간다.
change 이벤트에 대한 스크립트 작성 가능

컨디션패널(조회조건)
필수영역 및 옵션영역이 존재

버튼 패널
align = 버튼 정렬 속성
margin = 버튼 그룹 상단 여백

탭패널
다양한 설정 가능
tabType = 메인 서브 탭 중 선택옵션
selecteditem 초기화 시 선택되어 있는 탭 아이템 설정
cssClass 탭 패널에 css class명 정의
